# 📁 Google Drive Clone

A full-stack web application that replicates core functionalities of Google Drive — allowing users to upload, download, and manage files securely.

## 🚀 Features

- 🔐 User Authentication (Register / Login)
- ☁️ File Upload, Download, and Delete
- 🗂️ File metadata stored using MongoDB
- 📦 File storage handled by Supabase Storage
- 🧾 EJS templating engine for dynamic frontend views

## 🛠️ Tech Stack

- **Frontend:** EJS, HTML, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Cloud Storage:** Supabase
- **Authentication:** Custom middleware
- **File Handling:** Multer
